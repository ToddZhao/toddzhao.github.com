    Java基础语法

    第一个Java程序

    public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

    这个简单的程序输出「Hello, World!」。让我们分析一下这段代码：

    • public class HelloWorld：定义一个公共类，类名为HelloWorld。
    • public static void main(String[] args)：定义主方法，程序的入口点。
    • System.out.println("Hello, World!")：输出文本到控制台。

    基本数据类型

    Java有8种基本数据类型：

    • 整数类型：byte（8位）、short（16位）、int（32位）、long（64位）
    • 浮点类型：float（32位）、double（64位）
    • 字符类型：char（16位Unicode字符）
    • 布尔类型：boolean（true或false）
    int a = 10;
double b = 3.14;
char c = 'A';
boolean d = true;

    变量与常量

    变量是存储数据的容器，在Java中必须先声明变量的类型。

    int age;
age = 25;
// 或者
int age = 25;

    常量是不可改变的值，使用final关键字声明。

    final double PI = 3.14159;

    面向对象编程

    Java是一种面向对象的编程语言，它的核心概念包括类、对象、继承、多态和封装。

    类与对象

    类是对象的模板，定义了对象的属性和行为。对象是类的实例。

    public class Person {
    // 属性（成员变量）
    String name;
    int age;
    
    // 方法
    void speak() {
        System.out.println("My name is " + name + ", I am " + age + " years old.");
    }
}

// 创建对象
Person person = new Person();
person.name = "John";
person.age = 30;
person.speak();

    继承与多态

    继承允许一个类获取另一个类的属性和方法。多态允许不同类的对象对同一消息作出不同的响应。

    public class Animal {
    void makeSound() {
        System.out.println("Animal makes a sound");
    }
}

public class Dog extends Animal {
    @Override
    void makeSound() {
        System.out.println("Dog barks");
    }
}

public class Cat extends Animal {
    @Override
    void makeSound() {
        System.out.println("Cat meows");
    }
}

// 使用多态
Animal animal1 = new Dog();
Animal animal2 = new Cat();
animal1.makeSound();  // 输出：Dog barks
animal2.makeSound();  // 输出：Cat meows
